# 回测框架

## 介绍
回测是指基于历史已经发生过的真实行情数据，在历史上某一个时间点开始，严格按照编写的策略进行交易，模拟真实市场交易的规则进行买入、卖出，得出一个时间段内的收益情况。

#### 目标
评价策略的优劣

#### 举例
- 策略：跌破10元 -> 买入，高于15元 -> 卖出
- 标的：平安银行
- 时间：2012.01.01 - 2022.01.01
- 限制：T+1交易，即当日买入股票无法卖出
- 手续费率...
- ...

通过平安银行历史数据回测，便可得知该策略的一个表现。
